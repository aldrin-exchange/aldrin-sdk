## Instalation

```bash
npm install @aldrin/sdk
``` 
or 

```bash
yarn add @aldrin/sdk
```

## Development

1. Clone [repository](https://github.com/aldrin-exchange/aldrin-sdk)
2. Run `yarn` or `npm install`
